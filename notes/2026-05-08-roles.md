

Reporter
  - anyone basically can have this role

Triage, investigator
  - Reproduces bugs
  - Describes the issue in a clear repeatable way
  - Clarity? -> into the develment queue
  - Otherwise -> to PM??

Developer
  - takes bugs and features that have been marked "ready"
  - build the thing, on a worktree
  - get it reviewed and approved
  - get it aligned with main

Checker

Role: "release manager"
  - final code review, the final approver
  - decision on merge sequence once approved
  - enforce the deployment gate
  - get HITL approval
  - Give the operations manager a heads-up

Role: "operations manager"
  - Monitors production after a rollout
  - Investigates issues reported against the production system
